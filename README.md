# 🛡️ AI-Powered Cybersecurity Threat Detection Dashboard

## 🚀 Overview
This project is an AI-based cybersecurity system that detects network intrusions and anomalies using machine learning. It simulates a real-world Security Operations Center (SOC) by analyzing network traffic data and generating threat alerts.

---

## 🎯 Problem Statement
Traditional cybersecurity systems rely on rule-based detection, which fails to identify unknown or evolving threats. This project uses machine learning to automatically detect abnormal patterns in network traffic.

---

## 💡 Features
- 🔍 Anomaly Detection using Isolation Forest
- 📊 Interactive Dashboard (Streamlit)
- 🚨 Real-time Threat Alerts
- 📉 Confusion Matrix & Performance Metrics
- 📡 Network Traffic Simulation
- 💻 Terminal-style Alert Logs

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit

---

## 📂 Dataset
- NSL-KDD Dataset
- Contains simulated network traffic with attack labels:
  - DoS
  - Probe
  - R2L
  - U2R
